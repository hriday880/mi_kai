## Brand Name
- English: **Mi-KAI Tokyo** (stylized as **mi KAi TOKYO**)
- Kanji: **光明** (Kōmyō — Radiance / Brilliant Light)
- The logo uses a bold, geometric, blocky typeface with Japanese-influenced character forms
- "TOKYO" is set in clean uppercase tracking below
